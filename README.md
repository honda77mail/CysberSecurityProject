# CysberSecurityProject
The PDF is a cybersecurity project report about testing the OWASP Juice Shop application for vulnerabilities. It includes:

1. **Purpose**: Learn penetration testing by exploiting security flaws in the Juice Shop application.

2. **Findings**:
   - **Vulnerabilities**: SQL Injection, Cross-Site Scripting (XSS), brute force attacks, and hidden admin paths.
   - **Tools Used**: Burp Suite, OWASP ZAP, Dirb, Gobuster, and browser developer tools.

3. **Steps**: The report explains how vulnerabilities were exploited, such as:
   - Injecting malicious code.
   - Using SQL injection to log in as an admin.
   - Manipulating network requests for exploitation.

4. **Recommendations**:
   - Fix input validation issues.
   - Implement rate-limiting and CAPTCHA.
   - Regularly test and improve application security.

5. **Conclusion**: Highlights the application's high-risk vulnerabilities and suggests immediate fixes.
